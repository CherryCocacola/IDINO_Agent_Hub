## 1. 개요

### 1.1 시스템 목적

학기 스프린트 시스템은 학생들의 학업 및 진로 목표를 체계적으로 관리하고 추적하기 위한 기능입니다.
학과별, 학년별 특성에 맞는 맞춤형 목표를 제공하여 학생들의 효과적인 학업 계획 수립을 지원합니다.

### 1.2 데이터 범위

| 항목 | 값 |
|------|-----|
| 대상 입학년도 | 2023, 2024, 2025 |
| 총 대상 학생 수 | 7,436명 |
| 총 생성 목표 수 | 22,515건 |
| 학생당 평균 목표 | 3.0~3.1건 |

- 3학년(2023): 취업/자격증 중심 → high 비율 높음
- 2학년(2024): 심화학습/준비 → medium 비율 높음
- 1학년(2025): 기초/탐색 → low 비율 상대적 높음

## 4. 사용자 매뉴얼

### 4.1 학기 스프린트 접근 방법

#### 방법 1: 대시보드 AI 액션보드
1. 로그인 후 **대시보드** 메뉴 클릭
2. **AI 액션보드** 섹션에서 "학기 스프린트" 확인
3. 현재 학기 목표 요약 및 완료율 확인 가능

#### 방법 2: 스프린트 메뉴
1. 좌측 메뉴에서 **스프린트** 클릭
2. 칸반 보드 형태로 목표 관리 가능
3. 드래그 앤 드롭으로 상태 변경

### 4.2 칸반 보드 사용법

```
┌─────────────────────────────────────────────────────────────┐
│                    학기 스프린트 보드                          │
├───────────────┬───────────────┬───────────────┬─────────────┤
│   할 일 (Todo) │ 진행 중 (Doing)│ 완료 (Done)  │ 보류 (Hold) │
├───────────────┼───────────────┼───────────────┼─────────────┤
│ [목표 카드]    │ [목표 카드]    │ [목표 카드]  │ [목표 카드] │
│ - 제목        │ - 제목        │ - 제목       │ - 제목      │
│ - 우선순위    │ - 우선순위    │ - 우선순위   │ - 우선순위  │
│ - 설명        │ - 설명        │ - 설명       │ - 설명      │
└───────────────┴───────────────┴───────────────┴─────────────┘
```

#### 카드 이동
- 카드를 클릭하여 드래그
- 원하는 컬럼에 드롭하여 상태 변경

#### 우선순위 표시
- 🔴 **높음 (high)**: 빨간색 태그
- 🟡 **보통 (medium)**: 노란색 태그
- 🟢 **낮음 (low)**: 녹색 태그

### 4.3 새로고침 기능

스프린트 보드 우측 상단의 **새로고침 버튼(↻)** 클릭 시:
1. 서버에서 최신 목표 데이터 조회
2. localStorage 데이터 갱신
3. 화면 새로고침

> **참고**: localStorage에 저장된 로컬 변경사항은 새로고침 시 서버 데이터로 덮어씌워집니다.

### 4.4 목표 관리

#### 목표 추가 (현재 지원 안함)
- 칸반 보드에서 직접 추가 기능은 현재 미지원
- 관리자 또는 AI 코칭 시스템을 통해 추가

#### 목표 상태 변경
1. 카드 드래그 앤 드롭으로 컬럼 간 이동
2. 변경사항은 localStorage에 자동 저장

#### 목표 삭제 (현재 지원 안함)
- 칸반 보드에서 직접 삭제 기능은 현재 미지원

## 5. 관리자 매뉴얼

### 5.1 데이터베이스 구조

#### 테이블: `idino_career.tb_coaching_goal`

| 컬럼명 | 타입 | 설명 |
|--------|------|------|
| goal_id | SERIAL | 목표 고유 ID (PK) |
| std_id | VARCHAR | 학생 ID (FK → tb_student) |
| title | VARCHAR | 목표 제목 |
| description | TEXT | 목표 설명 |
| goal_type | VARCHAR | 목표 유형 (certification, academic, project, etc.) |
| priority | VARCHAR | 우선순위 (high, medium, low) |
| status | VARCHAR | 상태 (active, completed, draft) |
| progress_percentage | INTEGER | 진행률 (0~100) |
| target_date | DATE | 목표 달성 기한 |
| created_at | TIMESTAMP | 생성일시 |
| updated_at | TIMESTAMP | 수정일시 |

### 5.2 데이터 조회 쿼리

#### 학생별 목표 조회
```sql
SELECT g.*, s.student_nm, d.department_nm
FROM idino_career.tb_coaching_goal g
JOIN idino_career.tb_student s ON g.std_id = s.student_id
JOIN idino_career.tb_department d ON s.department_cd = d.department_cd
WHERE g.std_id = '학생ID'
ORDER BY
    CASE g.priority
        WHEN 'high' THEN 1
        WHEN 'medium' THEN 2
        ELSE 3
    END,
    g.created_at DESC;
```

#### 학년별 통계 조회
```sql
SELECT
    s.admission_year,
    COUNT(DISTINCT s.student_id) as student_count,
    COUNT(g.goal_id) as total_goals,
    ROUND(COUNT(g.goal_id)::numeric / COUNT(DISTINCT s.student_id), 2) as avg_goals
FROM idino_career.tb_student s
LEFT JOIN idino_career.tb_coaching_goal g ON s.student_id = g.std_id
WHERE s.admission_year IN (2023, 2024, 2025)
GROUP BY s.admission_year
ORDER BY s.admission_year;
```

#### 우선순위별 통계
```sql
SELECT priority, COUNT(*) as count,
       ROUND(COUNT(*) * 100.0 / SUM(COUNT(*)) OVER(), 1) as percentage
FROM idino_career.tb_coaching_goal
GROUP BY priority
ORDER BY
    CASE priority
        WHEN 'high' THEN 1
        WHEN 'medium' THEN 2
        ELSE 3
    END;
```
