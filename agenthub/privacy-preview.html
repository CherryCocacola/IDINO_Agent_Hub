1수집하는 개인정보 항목

회사는 서비스 제공을 위해 다음과 같은 개인정보를 수집합니다.

구분수집 항목필수 여부
회원가입 이메일 주소, 비밀번호(암호화), 이름 필수
프로필 부서, 직책, 연락처, 프로필 사진 선택
서비스 이용 에이전트 설정, 대화 내용, API 키, 사용 로그 자동 수집
결제 결제 수단 정보(카드 번호 일부), 결제 내역 결제 시 수집
기기 정보 IP 주소, 브라우저 종류, 운영체제, 접속 시간 자동 수집
비로그인 게스트(공개 에이전트 이용자)의 대화 내용은 서버에 저장되지 않으며 클라이언트 메모리에만 일시적으로 보관됩니다.

2개인정보 수집 방법

  • 직접 입력: 회원가입, 프로필 설정, 서비스 이용 시 회원이 직접 입력
  • 자동 수집: 서비스 이용 과정에서 자동으로 생성·수집 (접속 로그, 쿠키 등)
  • 제3자 로그인: 소셜 로그인(구글 등) 이용 시 해당 서비스로부터 제공받은 정보
  • 고객센터: 문의, 신고 접수 시 제공한 정보

5개인정보 제3자 제공

회사는 원칙적으로 이용자의 개인정보를 제3자에게 제공하지 않습니다. 다만, 다음의 경우는 예외입니다.

  • 이용자가 사전에 동의한 경우
  • 법령의 규정에 의하거나 수사기관의 적법한 요청이 있는 경우
  • 서비스 제공을 위해 업무 위탁이 필요한 경우 (아래 위탁 내역 참조)
회사는 개인정보를 수익 목적으로 제3자에게 판매하거나 임대하지 않습니다.

7AI 서비스 관련 개인정보 처리

AI 에이전트와의 대화 내용은 응답 생성을 위해 해당 AI 서비스 제공사(OpenAI, Anthropic, Mistral 등)의 API를 경유합니다. 이와 관련하여 다음 사항에 유의하세요.

  • AI 대화 내용은 각 서비스 제공사의 개인정보 처리방침의 적용을 받습니다.
  • 로그인 회원의 대화 내용은 서비스 개선을 위해 암호화하여 저장될 수 있습니다.
  • 비로그인 게스트의 대화는 서버에 저장되지 않습니다.
  • 개인 식별 정보, 금융 정보, 민감 정보는 AI에 입력하지 않도록 권고합니다.
AI 학습 데이터 활용 여부는 각 AI 제공사의 정책에 따릅니다. API 이용 약관상 비학습 처리가 기본 설정으로 적용됩니다.

9이용자의 권리

이용자는 언제든지 다음 권리를 행사할 수 있습니다.

열람권 본인의 개인정보 처리 현황을 확인할 수 있습니다.
정정권 부정확한 개인정보의 수정을 요청할 수 있습니다.
삭제권 개인정보의 삭제를 요청할 수 있습니다.
처리 정지권 개인정보 처리의 정지를 요청할 수 있습니다.
동의 철회권 선택적 수집에 대한 동의를 언제든지 철회할 수 있습니다.
이동권 본인 데이터를 구조화된 형태로 내려받을 수 있습니다.

권리 행사는 서비스 내 설정 페이지 또는 고객센터(support@agenthub.kr)를 통해 하실 수 있으며, 요청 후 10일 이내에 처리됩니다.

10개인정보 보안 조치

회사는 개인정보의 안전한 보관을 위해 다음과 같은 기술적·관리적 조치를 시행합니다.

  • 암호화: 비밀번호는 단방향 해시(bcrypt)로 암호화 저장. DB 데이터는 TLS 1.3으로 암호화 전송
  • 접근 통제: 최소 권한 원칙에 따른 DB 접근 제한, 내부 직원 접근 로그 관리
  • 취약점 관리: 정기적인 보안 점검 및 패치 적용
  • 침해 대응: 개인정보 침해 발생 시 72시간 이내 관계 기관 신고 및 이용자 통지
  • 물리적 보안: 클라우드 데이터센터의 물리적 보안 통제 적용

11아동 개인정보 보호

본 서비스는 만 14세 미만 아동을 대상으로 하지 않습니다. 만 14세 미만 아동의 개인정보가 수집된 사실이 확인되는 경우 즉시 삭제 조치합니다.

만 14세 미만의 경우 법정 대리인(부모 등)의 동의 없이 서비스에 가입하거나 이용할 수 없습니다.
