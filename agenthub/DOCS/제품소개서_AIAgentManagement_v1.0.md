## 1. 제품 개요

### 1.1 개요

AI Agent 통합관리 시스템(AIAgentManagement)은 기관·기업 내 다양한 인공지능(AI) 서비스를 단일 플랫폼에서 통합 운영·관리할 수 있도록 설계된 웹 기반 SaaS형 소프트웨어입니다.

본 시스템은 OpenAI(GPT), Anthropic(Claude), Google(Gemini), Perplexity, Mistral 등 주요 생성형 AI API를 하나의 인터페이스로 연결하고, 사용 목적에 맞는 AI Agent를 생성·배포·관리하는 기능을 제공합니다.

또한 RAG(Retrieval-Augmented Generation) 기반 지식 검색, 시각적 워크플로우 편집기, 멀티 에이전트 채팅, AI 이미지 생성, AI 프레젠테이션 생성 등 엔터프라이즈 AI 활용에 필요한 핵심 기능을 통합 제공합니다.

### 1.2 용도 및 목적

| 구분 | 내용 |
|---|---|
| **주요 용도** | 기관·기업 내 AI 서비스 통합 운영 플랫폼 |
| **목적** | 다양한 AI API를 단일 인터페이스로 관리하여 업무 효율화 및 AI 활용 비용 최적화 |
| **주요 사용자** | AI 서비스 운영 담당자, 일반 사용자, 시스템 관리자 |
| **도입 효과** | AI 서비스 도입 비용 절감, 조직 내 AI 활용 표준화, 사용량 모니터링 및 통제 |

**주요 활용 시나리오:**
- 부서별 전용 AI Agent 생성 및 배포 (인사, 법무, 고객응대 등)
- 사내 문서를 기반으로 한 AI 질의응답 서비스(RAG)
- 반복 업무 자동화를 위한 워크플로우 설계
- 이미지 생성, 프레젠테이션 자동 생성 등 창작 업무 지원
- 외부 서비스에 Agent 임베드(Embed) 제공

### 2.4 사용자 오류방지

본 제품은 사용자 입력 오류를 사전에 방지하기 위해 다음 메커니즘을 적용합니다.

#### 입력 유효성 검사

| 구분 | 적용 방식 |
|---|---|
| 필수 입력 항목 | `[Required]` 어트리뷰트로 누락 시 즉시 400 오류 반환 |

-- 1. 데이터베이스 생성
CREATE DATABASE AIAgentManagement;

.\iis-setting.ps1
```

# 1. 발행 (Publish)
dotnet publish -c Release -o ./publish

# 2. IIS 웹사이트 생성

#    - 앱 풀: .NET CLR 없음 (No Managed Code)

# 3. IIS 재시작
iisreset
```
