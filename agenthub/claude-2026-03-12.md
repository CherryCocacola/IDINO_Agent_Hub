### 추가된 Agent 모델 필드
```csharp
public string? WelcomeMessage { get; set; }   // 챗봇 환영 메시지 (max 500)
public string? PlaceholderText { get; set; }  // 입력창 안내 문구 (max 200)
public string? ChatTheme { get; set; }        // "light" | "dark" | "auto" (기본: "light")
public bool AllowGuestChat { get; set; }      // 비로그인 접근 허용 (기본: false)
```

> **DB 마이그레이션 필요**: 위 4개 컬럼이 Agents 테이블에 추가됨.
> EF Core migration 또는 아래 MSSQL로 직접 추가 가능:
> ```sql
> ALTER TABLE Agents ADD WelcomeMessage  NVARCHAR(500) NULL;
> ALTER TABLE Agents ADD PlaceholderText NVARCHAR(200) NULL;
> ALTER TABLE Agents ADD ChatTheme       NVARCHAR(10)  NULL DEFAULT 'light';
> ALTER TABLE Agents ADD AllowGuestChat  BIT           NOT NULL DEFAULT 0;
> ```

### 추가된 API 엔드포인트 (인증 불필요 — `[AllowAnonymous]`)
| Method | URL | 설명 |
|--------|-----|------|
| GET | `/api/agents/public/{code}/info` | 공개 Agent 정보 조회 |
| POST | `/api/agents/public/{code}/chat` | 퍼블릭 채팅 (비스트리밍) |
| POST | `/api/agents/public/{code}/stream` | 퍼블릭 SSE 스트리밍 채팅 |

**접근 조건**: `IsPublic = true` AND `IsActive = true` AND `AllowGuestChat = true`

### 신규 Vue 파일

#### `AgentPublicChat.vue` — 외부 공유 챗봇 페이지
- URL: `/chatbot/{agentCode}`
- 로그인 불필요, 공개 Agent만 접근 가능
- 기능: 환영 메시지 표시, 대화 이력 유지 (최근 10턴), 대화 초기화, 다크/밝은 테마
- 모바일 반응형 지원

#### `AgentEmbed.vue` — iframe 임베드 컴포넌트
- URL: `/embed/{agentCode}?theme=light|dark`
- iframe으로 외부 웹사이트에 삽입 목적
- 최소화/복원 버튼, `postMessage`로 부모 페이지와 통신
- 쿼리 파라미터로 테마 오버라이드 가능

#### `AgentTestPage.vue` — 내부 테스트 전용 페이지
- URL: `/agent-test/{agentCode}` (로그인 필요)
- 좌측: Agent 정보 + 공유 링크 (URL 복사 버튼 포함)
- 중앙: 채팅 인터페이스 + 응답 메타(모델명, 토큰, 응답시간)
- 우측: 디버그 패널 (토글) — 마지막 요청/응답 JSON, 총 토큰, 평균 응답시간

### ① 백엔드 API 테스트 (Swagger / curl)

**Agent 공개 정보 조회**
```http
GET /api/agents/public/{agentCode}/info
```
```bash
curl https://localhost:포트/api/agents/public/my-agent-1234/info
```
✅ 기대 응답:
```json
{
  "agentId": 1,
  "agentName": "...",
  "agentCode": "my-agent-1234",
  "welcomeMessage": "안녕하세요!",
  "placeholderText": "궁금한 점을 입력하세요...",
  "chatTheme": "light",
  "capabilities": ["chat", "stream"]
}
```
❌ `AllowGuestChat=false` 이면 404 응답

---

**퍼블릭 채팅 API**
```http
POST /api/agents/public/{agentCode}/chat
Content-Type: application/json

{
  "message": "안녕하세요",
  "messages": [
    { "role": "user", "content": "안녕하세요" }
  ]
}
```
```bash
curl -X POST https://localhost:포트/api/agents/public/my-agent-1234/chat \
  -H "Content-Type: application/json" \
  -d '{"message":"안녕하세요"}'
```
✅ 기대 응답:
```json
{
  "content": "안녕하세요! 무엇을 도와드릴까요?",
  "model": "gpt-4o",
  "tokensUsed": 42
}
```

---

**퍼블릭 SSE 스트리밍**
```bash
curl -X POST https://localhost:포트/api/agents/public/my-agent-1234/stream \
  -H "Content-Type: application/json" \
  -H "Accept: text/event-stream" \
  -d '{"message":"안녕"}'
```
✅ 기대 응답 (SSE):
```
event: start
data: {"agentId":1,"agentName":"..."}

event: chunk
data: {"content":"안녕 "}

event: chunk
data: {"content":"하세요! "}

event: done
data: {"totalTokens":30,"model":"gpt-4o"}
```
