## 1. 프로젝트 개요

**서비스명:** AI Agent Management System
**도메인:** agenthub.idino.co.kr
**목적:** 다양한 AI 서비스(OpenAI, Claude, Gemini 등)를 통합 관리하고, 사용자별 AI 에이전트·워크플로우·지식베이스를 운영하는 엔터프라이즈 플랫폼

## 12. Claude와 협업 시 참고사항

### 개발 환경
- **OS:** Windows 11 Enterprise
- **Shell:** bash (Unix 문법 사용)
- **현재 작업 디렉토리:** `C:\OneDrive\OneDrive - IDINO\01_업무\_조예주\_AI\CURSOR\AGENT_IDINO\AIAgentManagement`

### 개발 컨벤션 (확인된 사항)
- 백엔드: C# 표준 네이밍 컨벤션 (PascalCase 클래스/메서드, camelCase 변수)
- 프론트엔드: Vue 3 Composition API + TypeScript
- CSS: Bootstrap 5 유틸리티 클래스 + 커스텀 CSS
- API 통신: Axios 인터셉터 기반 JWT 자동 갱신

### 주요 개선 필요 영역 (파악된 사항)
- `PresentationBuilder.vue`: 63KB 대용량 파일 → 컴포넌트 분리 검토
- 워크플로우 빌더 UI/UX 보완
- 에이전트 마켓플레이스 디자인
- 반응형 레이아웃 세부 개선
- i18n 번역 완성도 향상

### Claude에게 작업 요청 시 권장 방식
1. 특정 파일/컴포넌트명 명시 (예: `AgentChat.vue`, `ChatService.cs`)
2. 기능 추가인지 버그 수정인지 구분
3. 디자인 변경 시 Bootstrap 컴포넌트 기반으로 요청
4. DB 스키마 변경 필요 시 EF Core 마이그레이션 방식 유지
